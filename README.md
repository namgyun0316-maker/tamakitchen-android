# 타마키친 (Tamakitchen)

냉장고 속 식재료를 관리하고
보유 재료를 기반으로 레시피를 추천하는 Android 앱입니다.

## 주요 기능

- 냉장고 식재료 관리
- 유통기한 관리
- 레시피 추천
- 장보기 리스트
- 공동 냉장고
- 펫 성장 시스템
- 도감 수집
- 합성 레시피
- 체크리스트 및 출석체크

## 기술 스택

### Android
- Java
- Android SDK
- RecyclerView
- Retrofit

### Backend
- Spring Boot
- JPA
- MySQL

### Infra
- Render
- Railway
- Cloudinary

### Analytics
- Firebase Analytics
