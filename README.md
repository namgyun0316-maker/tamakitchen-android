# 🍳 타마키친 (Tamakitchen)

Android(Java) + Spring Boot 기반의 1인 개발 프로젝트입니다.

냉장고 속 식재료를 관리하고,
보유한 재료를 기반으로 레시피를 추천하는 식재료 관리 애플리케이션입니다.

## ✨ 주요 기능

- 냉장고 식재료 관리
- 유통기한(D-Day) 관리
- 보유 재료 기반 레시피 추천
- 장보기 리스트
- 공동 냉장고
- 펫 성장 시스템
- 도감 수집
- 합성 레시피
- 체크리스트 및 출석체크

## 🛠 기술 스택
